# House Price Prediction

## Overview

This project develops a machine learning-based system for predicting residential property prices using housing and location-related features.

The objective is to estimate property values accurately and support data-driven decision-making in the real estate sector.

## Problem Statement

Accurate property valuation is essential for buyers, sellers, and real estate investors.

This project aims to predict house prices using machine learning regression techniques and housing market data.

## Dataset

A housing dataset was used containing property-related and location-based features.

### Features Included

- Property Area
- Number of Rooms
- Property Age
- Location Score
- House Price
